What is the Service Layer?

The service layer in the Absence Management System handles all communication between the frontend and backend API. It provides a clean, consistent interface for components to fetch data, submit updates, and manage application state.

This layer abstracts away the complexities of API calls, error handling, and data formatting, allowing the component layer to focus on rendering UI and user interactions rather than data fetching logic.
